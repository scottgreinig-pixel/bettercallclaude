Eval Set Review: legal-briefing

Current description: Proactively suggests or activates a structured briefing session before executing complex Swiss legal queries. Use this skill whenever a legal query spans multiple domains, is ambiguous about output or jurisdiction, involves multi-agent coordination, mentions financial exposure above CHF 100,000, or uses open-ended verbs like 'handle', 'deal with', 'advise on'. Do NOT wait for the user to ask for a briefing — detect complexity early and suggest it before routing. Activate for litigation-prep, due-diligence, contract-lifecycle, multi-jurisdictional, FINMA/AML, M&A, employment+social insurance, or any query where skipping intake could send the wrong agents down the wrong path.
